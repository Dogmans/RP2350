# RP2350 Documentation & Programs

This repository provides:

- **Documentation** for the Waveshare RP2350-LCD-1.28 and related hardware, including quick reference guides, demo maps, and usage instructions.
- **Example Programs** for the RP2350-LCD-1.28, covering Arduino, C, and Python implementations for display and sensor interfacing.
- **PC Stats Display Project**: A host/device system for showing PC stats (CPU/GPU temp, FPS, RAM, etc.) on the RP2350 circular LCD, with configurable layouts and colors.

## Structure

- `docs/` — Guides, quick references, and demo maps for RP2350 hardware.
- `pc_stats/` — Host and device code for PC stats display, including configuration and deployment scripts.
- `waveshare-rp2350/` — Example code for Arduino, C, and Python for the RP2350-LCD-1.28.

## Getting Started

- See `pc_stats/README.md` for instructions on running the PC stats display project.
- See `docs/waveshare-rp2350/` for hardware documentation and quick reference.

## Purpose

This repo is intended as a central resource for:
- Users of the Waveshare RP2350-LCD-1.28
- Developers building display and sensor projects
- Anyone wanting to show PC stats on a circular LCD

Contributions and feedback are welcome!
